# AutoFlow Control Center 更新指南

本指南將協助你輕鬆更新 AutoFlow Control Center 到最新版本。

---

## 🚀 快速開始 - 應用內更新

### 方法一:一鍵更新腳本 (推薦,最快速)

這是最簡單的更新方式,**無需重啟程式**即可生效!

1. **開啟更新檢查**
   - 點擊側邊欄底部的「檢查更新」按鈕
   - 或等待程式啟動時自動檢查

2. **選擇快速更新**
   - 在更新對話框中,點擊「更新腳本」按鈕
   - 等待下載完成(通常只需幾秒鐘)

3. **完成!**
   - 更新會立即生效,無需重啟程式
   - 你的 Excel 轉換和截圖功能已更新到最新版本

> [!TIP]
> **適用情況**: 當更新僅涉及外部腳本(`excel_轉換/` 或 `截圖腳本/`)時,使用此方法最快速方便。

---

## 📦 完整更新步驟

當核心程式有重大更新時,需要下載完整更新包。

### 方法二:應用內下載 (半自動)

1. **檢查更新**
   - 點擊側邊欄的「檢查更新」按鈕

2. **下載完整更新包**
   - 在更新對話框中,點擊「下載完整更新包」
   - 等待下載完成(約 50-100 MB,視網速而定)
   - 下載完成後,程式會自動開啟下載資料夾

3. **解壓並覆蓋**
   - 關閉 AutoFlow Control Center
   - 解壓縮下載的 `.zip` 檔案
   - 將解壓後的所有檔案**覆蓋**到原程式目錄

4. **重新啟動**
   - 執行新版本的 `.exe` 檔案
   - 完成更新!

### 方法三:手動下載 (傳統方式)

1. **前往 GitHub Releases**
   - 訪問 [Releases 頁面](https://github.com/kevin-leeeeee/auto_screenshot/releases)
   - 或在更新對話框中點擊「前往 GitHub Releases」

2. **下載最新版本**
   - 找到最新的 Release
   - 下載 `AutoFlow_Control_Center_vX.X.X_Full.zip`

3. **解壓並覆蓋**
   - 關閉 AutoFlow Control Center
   - 解壓縮下載的檔案
   - 覆蓋到原程式目錄

4. **執行新版本**
   - 雙擊新版本的 `.exe` 檔案

## 🔧 進階選項

### 手動更新特定元件

如果你只想更新特定的腳本檔案:

1. **前往 GitHub 原始碼**
   ```
   https://github.com/kevin-leeeeee/auto_screenshot
   ```

2. **下載特定檔案**
   - 導航到 `excel_轉換/` 或 `截圖腳本/` 目錄
   - 下載你需要的 `.py` 檔案

3. **手動替換**
   - 將下載的檔案覆蓋到對應的資料夾
   - 無需重啟程式

### 回退到舊版本

如果新版本有問題,你可以回退:

1. **找到備份**
   - 更新腳本時,程式會自動建立備份
   - 備份檔案位於 `excel_轉換/` 或 `截圖腳本/` 目錄
   - 檔名格式: `檔名.py.bak.時間戳記`

2. **還原備份**
   - 刪除或重新命名當前的 `.py` 檔案
   - 將備份檔案重新命名為原始檔名
   - 例如: `convert_excel.py.bak.1705123456` → `convert_excel.py`

## ❓ 常見問題

### Q: 更新後程式無法啟動怎麼辦?

**A:** 請嘗試以下步驟:
1. 確認目錄結構完整:
   ```
   AutoFlow_Control_Center_vX.X.X/
   ├── AutoFlow_Control_Center_vX.X.X.exe
   ├── excel_轉換/
   └── 截圖腳本/
   ```
2. 重新下載完整更新包
3. 解壓到新的資料夾,不要覆蓋舊版本
4. 如果問題持續,請回報 [GitHub Issues](https://github.com/kevin-leeeeee/auto_screenshot/issues)

### Q: 更新腳本失敗怎麼辦?

**A:** 可能的原因和解決方法:
- **網路連線問題**: 檢查網路連線,稍後再試
- **GitHub 連線失敗**: 使用「手動下載」方式
- **檔案權限問題**: 以系統管理員身分執行程式

### Q: 如何確認目前版本?

**A:** 查看側邊欄頂部的版本號,例如 `v2.1.0`

### Q: 更新會遺失我的設定嗎?

**A:** 不會!你的設定儲存在 `app_data.json`,更新不會影響:
- 自動化設定
- 執行歷史
- 統計數據
- 顯示偏好

### Q: 可以跳過某個版本嗎?

**A:** 可以,但建議始終更新到最新版本以獲得:
- 最新功能
- 錯誤修復
- 效能改善
- 安全性更新

### Q: 更新需要多久?

**A:** 
- **快速更新(腳本)**: 5-10 秒
- **完整更新(下載)**: 1-5 分鐘(視網速而定)
- **完整更新(解壓覆蓋)**: 1-2 分鐘
